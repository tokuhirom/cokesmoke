# 変更履歴

## 未リリース

### 新機能
- スキル習得制限: 職業ごとに習得可能なスキルを制限（例: 戦士はバリアのみ、魔法使いはヒール不可）
- 装備制限システム: 職業ごとに装備可能な武器タイプを制限（狩人は短剣のみ）
- 矢システム: 射撃スキルが矢（消耗品）を消費するように変更
  - 木の矢（攻撃力80%）: 木材×1 → 5本
  - 鉄の矢（攻撃力120%）: 木材×1 + 鉄鉱石×1 → 5本
  - 狩人は木の矢10本を持ってスタート
- 短剣を追加: 木の短剣（攻撃+1）、鉄の短剣（攻撃+3）
- 斧を追加: 鉄の斧（攻撃+7）、ミスリルの戦斧（攻撃+15 防御-2）— 戦士専用
- 消耗品クラフトシステム: 商人で矢などの消耗品を作成可能
- ダンジョンマップ拡大: 35×20 → 70×40 にサイズ倍増、カメラがプレイヤーに追従してスクロール
- ダンジョン固定化: 地形レイアウトがワールドごとに固定（敵・アイテムは毎回ランダム）
- ダンジョン内デスドロップ: 死亡時に装備中のアイテムは保持、未装備品はダンジョン内にドロップ
- 視覚的警告: 空腹0でステータスバー赤点滅、満腹20以下でオレンジ表示、HP10%以下で黄色表示
- ダンジョン内セーブ/復帰: ダンジョン中の状態がセーブされ、リロードしても同じ階に復帰（リロード脱出を防止）

### 変更
- 射撃スキルのMP消費を0に変更（矢の消費がコスト）
- 戦士の装備制限を剣・斧のみに変更
- 武器にweaponType属性を追加（sword, dagger, staff, axe, spear）
- ランダムアーティファクト武器にもweaponTypeを付与

## 0.1.0

### 新機能
- 射撃スキルを自動ターゲットに変更（視界内の最も近い敵を攻撃）
- ダンジョンに罠システムを追加（落とし穴・矢の罠・毒ガス・空腹の罠）
- 始まりの迷宮を始まりの村の近くに配置（距離4-8）
- UIをメニューボタンに統合、ステータス画面とタイトル戻り動線を追加
- ゴールドシステムと全村に宿屋を追加
- 職業システム（戦士・魔法使い・僧侶・盗賊・狩人）
- スキルスロット満杯時のポケモン式入れ替えメニュー
- 回復魔法ヒール（HP25回復）とヒールII（HP60回復）
- 消耗品インベントリシステム（ポーション等を保持して任意タイミングで使用）
- 木材・獣の皮素材とクラフトレシピ
- 商人に装備解体機能
- 女神の贈り物システム（炎耐性・毒無効等）
- 属性耐性・ランダムアーティファクト
- 装備インベントリとダンジョンごとのボス
- ワールドマップ・街・ダンジョン・仲間システム
- セーブ/ロード（複数ワールドスロット）
- 装備・素材・クラフト・NPC勧誘
- 空腹システム（燃料システムから置き換え）
- 異世界転生テーマのプロローグとチュートリアル
- 8方向移動対応
