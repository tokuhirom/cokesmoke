## 世界観

蒸気文明が極限まで発達した地下都市。上層の貴族街に続く坑道を、蒸気義肢を持つ探索者が進む。  
燃料（石炭）が尽きると蒸気義肢が止まり、動けなくなる＝死。

---

## コアメカニクス

### 蒸気圧システム

- 全行動（移動・攻撃・スキル）は**蒸気圧（SP）**を消費する
- SPはターン経過で自動回復（ボイラーが温まる）
- SP上限を超えた状態で行動すると**過負荷（Overload）**が発生し、爆発ダメージ
- SP = 攻撃力と防御力にも影響（高圧ほど強いが危険）

```
SP: [====----] 40/80  ← 残量と上限
```

### 燃料システム

- 探索者は**石炭**を消費しながら動く（ターンごとに-1）
- 石炭切れ → 蒸気義肢停止 → 即死
- ダンジョン内の燃料缶・石炭袋を拾いながら進む
- 緊張感の根幹

### 視界（FOV）

- ROT.js の PreciseShadowcasting を使用
- 視界半径: 5マス（ランタンアイテムで拡大可能）
- 既訪問マスは薄く表示（霧の中に消える演出）

## ゲームループ

```
ダンジョン入口
    ↓
フロア探索（ターン制）
    ├── プレイヤー入力待ち
    │       ↓
    │   移動 / 攻撃 / アイテム使用 / 待機
    │       ↓
    │   敵ターン処理
    │       ↓
    │   描画更新
    │       ↓
    └── 繰り返し
    ↓
階段発見 → 次フロアへ
    ↓
ボス撃破 → クリア / 死亡 → ゲームオーバー
```

- フロア数: **10階層**（1ラン 20〜40分目安）
- ボス: 5階・10階に配置

## ダンジョン生成

ROT.js の **Digger** アルゴリズムを使用。

```
████████████████████
█    █    ≡≡≡█     █
█ @  █         █  █
█    █    ☿    █  █
████████████████████
```

| 記号 | 意味 |
|------|------|
| `█` | 壁（鉄板） |
| ` ` | 床 |
| `≡` | 蒸気管（踏むとSP+回復） |
| `#` | 鉄格子（視線は通るが移動不可） |
| `>` | 階段（次フロアへ） |
| `@` | プレイヤー |
| `☿` | アイテム |
| `Ω` | 敵 |

---

## 敵

| 記号 | 名前 | 特徴 |
|------|------|------|
| `g` | ゴーレム兵 | 低速・高耐久 |
| `s` | 蒸気スパイダー | 高速・群れで行動 |
| `B` | ボイラーブルート | SP過負荷で爆発する |
| `W` | 霧の番人（ボス） | 視界を奪う霧を展開 |

敵AIはROT.js の `Path.AStar` で経路探索。  
プレイヤーが視界外なら徘徊、視界内なら追跡。

---

## アイテム

| 記号 | 名前 | 効果 |
|------|------|------|
| `♦` | 石炭袋 | 燃料+30 |
| `!` | 圧力弁 | SP上限+20（永続） |
| `?` | 設計図 | ランダムスキル習得 |
| `/` | 蒸気ランス | 攻撃力+（装備） |
| `[` | 鉄板鎧 | 防御力+（装備） |
| `~` | 潤滑油 | SP回復速度+1ターン |

---

## スキルシステム

設計図（`?`）を使うとスキルを1つ習得。最大3スロット。

| スキル名 | 効果 | SPコスト |
|----------|------|---------|
| 蒸気噴射 | 前方2マスにダメージ | 20 |
| 緊急弁開放 | SP全消費→即時大ダメージ | 全SP |
| 装甲展開 | 1ターン無敵 | 15 |
| 煙幕 | 周囲を霧で覆う（逃走用） | 10 |

## UI レイアウト（スマホ縦画面）

```
┌─────────────────────┐
│ HP:████░░  燃料:48  │  ← ステータスバー
│ SP:[=====---] 50/80 │
├─────────────────────┤
│                     │
│   ダンジョン表示    │  ← メインエリア（ROT.js Canvas）
│   （ASCII）         │
│                     │
├─────────────────────┤
│ [スキル1][スキル2]  │  ← スキルボタン
│ [スキル3][待機]     │
└─────────────────────┘
```

- 移動: **スワイプ**（4方向）または**タップ**（移動先指定）
- スキル: 画面下部ボタン
- アイテム: 自動拾得 or 長押しメニュー
